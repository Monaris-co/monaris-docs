## V1 (current build focus)

**Theme:** Make existing product behavior trustworthy and sticky.

### Sprint 1 (weeks 1–2)

- Volume minimums per tier (system-enforced)
- Score explanation breakdown
- Tier labels in plain language
- "Coming soon" copy for credit terms until launch

### Sprint 2 (weeks 3–4)

- Mona event messaging (5 triggers)
- Cashflow picture (wallet connect, timeline, concentration flag)

### Sprint 3 (weeks 5–6)

- Shareable Score card + verify link
- Score improvement tips (action + approximate point impact)

---

## V2 (after retention milestone)

Start V2 once V1 reaches stable weekly retention.

### Core V2 scope

- Private payments via Privacy Router (default private)
- Programmable privacy rules
- Selective disclosure proofs
- Public Score profile
- Off-chain connectors (Plaid, QuickBooks/Xero, CSV fallback)
- Score API (soft launch, consent-gated)
- Mona conversational interface
- Credit pre-approval display only (no disbursement)
